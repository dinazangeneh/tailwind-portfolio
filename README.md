# 🌐 Personal Portfolio Website

A modern, responsive personal portfolio website built with **HTML5** and **Tailwind CSS**.
This project showcases my skills, projects, and experience through a clean, responsive, and user-friendly interface.

## ✨ Features

* Responsive design for desktop, tablet, and mobile
* Modern and clean UI design
* Built with Tailwind CSS
* Smooth navigation
* Projects showcase section
* Skills and experience section
* Contact section
* Optimized and lightweight structure

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/yourusername/tailwind-portfolio.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No installation or additional dependencies are required.
